### How to run the project locally?
If you want to run the project locally, follow the bellow steps

* first clone the project 
   ```
   git clone https://github.com/s27840213/Infinite-Scroll.git
  ```
* install the dependencies
  ```
  npm install
  ```
* execute the project
   ```sh
  npm run serve
  ```
